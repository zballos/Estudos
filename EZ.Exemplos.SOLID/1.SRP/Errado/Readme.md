Perceba que na classe Pedido, desta sessão, ela executa três tipos diferentes de tarefas. 

- Tratar dados do Pedido, como Produtos (Adicionar Produto / Calcular Total de Produtos / Adicionar ou Remover Produto);
- Exibição e impressão do Pedido (Montrar / Imprimir);
- Manipulação do Pedido (Obter / Salvar / Excluir / Atualizar Produto)
